迭代器

  • 迭代
    我们用一个循环来遍历容器中的元素时，这种遍历过程就叫迭代
  • 可迭代对象
    含有__iter__()方法或__getitem__()方法的对象称之为可迭代对象

可见，字典、元组、集合和字符串对象都是可迭代的
迭代器是指遵循迭代器协议的对象。

是指要实现对象的iter() 和next()方法，其中,__iter()__方法返回迭代器对象本身，__next__()方法返回容器的下一个元素，在没有后续元素时抛出StopIteration异常。
